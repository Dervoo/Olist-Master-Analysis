# Moduł Python dla Analityka Biznesowego

Ten folder zawiera warstwę programistyczną projektu, rozszerzającą analizy SQL i Power BI o modelowanie danych, prognozowanie i interaktywne raportowanie.

## Struktura:
- `scripts/`: Gotowe skrypty analityczne i automatyzacyjne.
- `notebooks/`: Eksperymentalna analiza danych (Jupyter Notebooks).
- `streamlit_app/`: Kod interaktywnego dashboardu biznesowego.

## Instalacja:
Aby zainstalować wymagane biblioteki, uruchom:
```bash
pip install -r requirements.txt
```
